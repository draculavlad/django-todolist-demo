## Run Backend

### Usage

- cd api
- pip install requirements.txt
- python manage.py migrate
- python manage.py runserver 0.0.0.0:8000

## Run FrontEnd

### Usage

- cd web
- npm install
- npm run dev
- visit http://127.0.0.1:8080
